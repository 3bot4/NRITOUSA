- **No ESLint config** (`.eslintrc*` / `eslint.config.*` / `package.json#eslintConfig` all absent). `next lint` is therefore non-functional in CI. Add `.eslintrc.json` with `{ "extends": "next/core-web-vitals" }`.

- **Client/server correctness:** 0 `page.tsx` files use `"use client"`, so there is **no `metadata` exported from a client component** and no client/server metadata misuse. `generateMetadata`/`export const metadata` are used correctly on server pages.
- **No duplicate `pageMetadata` patterns**, no invalid dynamic-route usage found; every `[slug]` route has a matching `generateStaticParams`.
- **Stale `.next`:** build cache from Jul 5, no `BUILD_ID` — harmless locally; a clean build on deploy supersedes it.

## Sitemap / Robots Issues

**Sitemap** (`src/lib/sitemap-data.ts`, split into `/sitemap.xml` index + pages/tools/articles/tax/immigration):
- **Missing:** `/partnerships` is a real indexable page but is not in any sitemap segment. Add it (or noindex if intentional).
- **No stale entries:** every literal `e("/…")` maps to a real route; the 12 apparent "stale" paths were `${slug}` template builders (dynamic families), all correctly covered.
- **Intentional exclusions look correct:** redirects (`/privacy`, `/terms-of-use`, `/topics/money-transfer`, `/tools/nri-global-wealth-tax-organizer`), noindex app-states, and `/api/*` are excluded by design.
- **Minor:** author pages (`/author/*`) are excluded from the sitemap but are **indexable** (no `noindex`) — pick one signal.

**Robots** (`src/app/robots.ts`):
- **Gap (P0):** `disallow` includes `"/*/opengraph-image"` and `"/*/opengraph-image?*"` — these match **nested** OG routes (e.g. `/articles/foo/opengraph-image`) but **not** the root `"/opengraph-image"` (the `*` requires a segment before it). Result: Googlebot crawls the site-wide OG card, which is exactly the two `/opengraph-image?<hash>` URLs appearing under "Crawled – currently not indexed" in Search Console. **Fix:** add `"/opengraph-image"` and `"/opengraph-image?*"` to the array.
- Correct otherwise: `/api/` disallowed; sitemap URL is the canonical `https://www.nritousa.com/sitemap.xml`; no localhost/staging/old-domain references anywhere.
